Location
Represents a geographical location with optional latitude and longitude. This model is used to store location data, such as for stops or stations. The type field is set to "location" to indicate the nature of this model. The id field is a unique identifier for the location. The latitude and longitude fields are optional and can be used to specify the exact geographical coordinates of the location.
