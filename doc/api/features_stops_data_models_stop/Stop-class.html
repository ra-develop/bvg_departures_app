Stop

Stop class

Represents a stop in the BVG system. This model includes the type of stop, its ID, name, location, weight, relevance, score, and a list of stops. The type field is set to "stop" by default, indicating that this model is primarily used for stops. The id field is a unique identifier for the stop, which can be an IBNR or similar. The name field is the name of the stop. The location field contains geographical coordinates (latitude and longitude). The weight, relevance, and score fields are used for sorting or ranking stops. The stops field is a list of other stops that may be related to this stop. The products field contains information about the available transportation products. This model is serializable to and from JSON format, making it suitable for use in APIs or local storage.
