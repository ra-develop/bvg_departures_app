Provider<StopRepository> stopRepositoryProvider
final

Provider for the StopRepository. This provider is used to access the repository methods for fetching stops. It allows the application to retrieve stop data from the BVG API. The repository abstracts the data layer, making it easier to manage dependencies and test the application.

Implementation
