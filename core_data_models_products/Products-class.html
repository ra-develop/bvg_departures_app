Products class

Represents the different types of transportation products available. This model is used to indicate which transportation products are available for a specific stop or station. /// The fields are boolean values indicating the availability of each product. The suburban, subway, tram, bus, ferry, express, and regional fields represent the respective transportation products. All fields are optional and default to false if not provided. This model is serializable to and from JSON format, making it suitable for use in APIs or local storage.
